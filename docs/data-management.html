5 Data Management and Preparation in R

5.1 Understanding Your R Environment

Before we dive into data analysis, it’s essential to understand how to manage your R environment effectively. This includes setting up your workspace, understanding file paths, and organizing your analytical projects.

5.1.1 The RStudio Interface

RStudio provides four main panes that help you work efficiently:

  1. Source Editor (top-left): Where you write and edit R scripts
  2. Console (bottom-left): Where R executes commands and displays output
  3. Environment/History (top-right): Shows loaded datasets and command history
  4. Files/Plots/Packages/Help (bottom-right): File browser, plot viewer, and help documentation
# You can run code from the Source Editor by:
# 1. Placing cursor on the line and pressing Ctrl+Enter (Windows) or Cmd+Enter (Mac)
# 2. Selecting multiple lines and pressing Ctrl+Enter or Cmd+Enter
# 3. Clicking the "Run" button in RStudio

5.1.2 Working Directory Management

Your working directory is the folder where R looks for files and saves output by default.

# Check your current working directory
getwd()
## [1] "/home/sean/Documents/claude/Course Books/SPM 4703"
# Set a new working directory (modify path as needed)
# setwd("~/Documents/Sport_Analytics")  # Mac/Linux
# setwd("C:/Users/YourName/Documents/Sport_Analytics")  # Windows

Best Practice: Create a dedicated folder for each R project and set it as your working directory at the start of each session.

5.2 Installing and Loading Packages

R’s functionality extends through packages - collections of functions, data, and documentation created by the R community.

5.2.1 Installing Packages

You only need to install a package once on your computer:

# Install the tidyverse suite of packages
install.packages("tidyverse")

# Install ggplot2 for data visualization
install.packages("ggplot2")

# Install multiple packages at once
install.packages(c("dplyr", "readr", "tidyr"))

5.2.2 Loading Packages

After installation, you must load packages in each R session before using them:

# Load tidyverse (includes dplyr, ggplot2, readr, and more)
library(tidyverse)

# Load ggplot2 specifically
library(ggplot2)

Important: You install once but load every session.

5.3 Loading and Exploring Data

5.3.1 Reading CSV Files

CSV (Comma-Separated Values) files are the most common format for sport analytics data.

# Basic method - assumes file is in working directory
data <- read.csv("Chapter 2 data.csv")

# Using file.choose() - opens file browser dialog
data <- read.csv(file.choose())

# Using full file path
data <- read.csv("~/Documents/Sport_Analytics/Chapter 2 data.csv")

5.3.2 Exploring Dataset Structure

Once data is loaded, explore its structure using these essential functions:

# Display first 6 rows
head(data)

# Display last 6 rows
tail(data)

# Show data structure and types
str(data)

# Get summary statistics for all variables
summary(data)

# Display column names
colnames(data)

# Check number of rows
nrow(data)

# Check number of columns
ncol(data)

# Get dimensions (rows, columns)
dim(data)

5.4 Understanding Data Types in R

# Numeric (numbers with decimals)

class(num_athletes)
## [1] "integer"
# Character (text)
school_name <- "University of North Florida"
class(school_name)
## [1] "character"
# Logical (TRUE/FALSE)
is_power5 <- TRUE

## [1] "logical"
# Factor (categorical data)
conference <- factor(c("SEC", "ACC", "Big Ten", "SEC"))
class(conference)

5.5 Checking for Data Quality Issues

5.5.1 Identifying Missing Values

Missing data (represented as NA in R) is common in real-world datasets:

# Create sample data with missing values
sample_data <- data.frame(
  player = c("Player A", "Player B", "Player C", "Player D"),
  points = c(22, 18, NA, 25),
  assists = c(5, NA, 8, 6),
  rebounds = c(4, 6, 3, NA)
)

# Check for missing values
is.na(sample_data)
##      player points assists rebounds
## [1,]  FALSE  FALSE   FALSE    FALSE
## [2,]  FALSE  FALSE    TRUE    FALSE
## [3,]  FALSE   TRUE   FALSE    FALSE
## [4,]  FALSE  FALSE   FALSE     TRUE
# Count total missing values
sum(is.na(sample_data))
## [1] 3
# Count missing values per column
colSums(is.na(sample_data))
##   player   points  assists rebounds
