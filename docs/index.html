About This Textbook

Welcome! This book provides a comprehensive introduction to data analysis and visualization in the sports industry using the R programming language. Whether you’re interested in working for a professional sports team, analyzing athletic department finances, or understanding fan engagement metrics, this book will equip you with the essential analytical skills needed in today’s data-driven sport business landscape.

Why Sport Business Analytics?

The sport industry has undergone a dramatic transformation in recent years. Organizations now rely on data-driven insights to:

  • Optimize player performance and scouting decisions
  • Maximize ticket sales and revenue
  • Enhance fan engagement and marketing strategies
  • Improve facility operations and event management
  • Make strategic business decisions backed by evidence

This course prepares you to be part of this exciting evolution by teaching you how to collect, analyze, visualize, and communicate insights from sport-related data.

About This Book

This textbook is designed for SPM 4703 - Sport Business Analytics at the University of North Florida. The book combines theoretical foundations with hands-on practical applications, preparing you for careers in sport analytics, business intelligence, and data-driven decision making in the sports industry.

What You’ll Learn

Throughout this book, you will:

  • Master the fundamentals of R programming and RStudio
  • Collect and manage sport-related data from various sources
  • Apply statistical methods to analyze sport business problems
  • Create compelling data visualizations using R and Tableau
  • Conduct descriptive and predictive analytics
  • Communicate findings effectively to stakeholders
  • Complete a real-world sport analytics project

Course Structure

The book is organized to match the progression of topics in SPM 4703:

Part I: Foundations Introduction to sport analytics, R programming basics, and data collection methods

Part II: Research Methods Sampling techniques, validity, reliability, and research design

Part III: Data Analysis Descriptive statistics, statistical testing (t-tests, ANOVA), correlation, and regression

Part IV: Visualization Data visualization principles using R (ggplot2) and Tableau

Part V: Applications Case studies and real-world applications in sport business

Prerequisites

This book assumes:

  • Basic understanding of sports and sport business operations
  • No prior programming experience (we’ll teach you R from scratch!)
  • Basic statistics knowledge is helpful but not required
  • Access to a computer with R and RStudio installed

Software Requirements

You will need to install the following free software:

  1. R (version 4.0 or higher): Download from CRAN
  2. RStudio Desktop: Download from Posit
  3. Tableau Public (free version): Download from Tableau

How to Use This Book

Each chapter includes:

  • Conceptual explanations of key topics
  • R code examples that you can run yourself
  • Practice exercises to reinforce learning
  • Real sport data from sources like the Lahman baseball database
  • Case studies from professional sports organizations

Important: All code in this book is designed to be copied and executed in your own R environment. We encourage you to:

  1. Type out the code yourself (don’t just copy-paste) to build muscle memory
  2. Experiment by modifying the code to see what happens
  3. Complete all practice exercises
  4. Work through examples with your own sport data when possible

Dataset Information

Throughout the book, we’ll work with several datasets:

  • NCAA Division I athletic department financial data
  • Lahman baseball database (historical MLB statistics)
  • Custom sport business datasets (ticket sales, attendance, etc.)
  • Sample data for practice exercises

All datasets are either: - Included in R packages - Available for download from the course Canvas site - Fictional data created for educational purposes

Getting Help

If you encounter issues:

  1. Check the relevant chapter’s troubleshooting section
  2. Review error messages carefully - they often tell you what’s wrong
  3. Use R’s built-in help: ?function_name or help(function_name)
  4. Search online - R has an excellent community
  5. Ask during class or office hours
