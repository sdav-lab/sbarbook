2 Getting Started with R for Sport Analytics

2.1 Installing R and RStudio

Before we dive into sport business analytics with R, we need to set up our computing environment. R is a free, open-source programming language specifically designed for statistical computing and data analysis.

2.1.1 R Installation

R can be downloaded from the Comprehensive R Archive Network (CRAN) at https://cran.r-project.org/.

For Mac users: 1. Visit https://cran.r-project.org/bin/macosx/ 2. Download the latest .pkg file 3. Double-click to install

For Windows users: 1. Visit https://cran.r-project.org/bin/windows/base/ 2. Download the latest installer 3. Run the installer

For Linux users: 1. Use your distribution’s package manager 2. Or download from https://cran.r-project.org/

2.1.2 RStudio Installation

RStudio is the recommended integrated development environment (IDE) for R:

  1. Visit https://www.rstudio.com/products/rstudio/download/
  2. Download the appropriate version for your operating system
  3. Install and launch RStudio

2.2 R Basics

R is a powerful statistical programming language. Understanding basic R concepts is essential for sport analytics work.

2.6 Data Management Best Practices

2.6.1 Handling Missing Data

# Create sample data with missing values
sample_data <- data.frame(
  player = c("Player A", "Player B", "Player C", "Player D"),
  points = c(22, 18, NA, 25),
  assists = c(5, NA, 8, 6),
  rebounds = c(4, 6, 3, NA),
  stringsAsFactors = FALSE
)

# Check missing values
is.na(sample_data)
##      player points assists rebounds
## [1,]  FALSE  FALSE   FALSE    FALSE
## [2,]  FALSE  FALSE    TRUE    FALSE
## [3,]  FALSE   TRUE   FALSE    FALSE
## [4,]  FALSE  FALSE   FALSE     TRUE
# Remove rows with missing data
complete_cases <- sample_data[complete.cases(sample_data), ]
complete_cases
##     player points assists rebounds
## 1 Player A     22       5        4
# Replace missing values with 0
cleaned_data <- sample_data
cleaned_data[is.na(cleaned_data)] <- 0
cleaned_data
##     player points assists rebounds
## 1 Player A     22       5        4
## 2 Player B     18       0        6
## 3 Player C      0       8        3
## 4 Player D     25       6        0
